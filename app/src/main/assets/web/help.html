ကျမ်းစာ လမ်းညွှန်

ပါဠိ၊ အဋ္ဌကထာ၊ ဋီကာ နှင့် အည ဆိုပြီး လေးကဏ္ဍ ခွဲခြားထားပေးပါသည်။

ပါဠိ၊ အဋ္ဌကထာ၊ ဋီကာ ကဏ္ဍတို့၌ ရွေးချယ်ရ လွယ်ကူစေရန် ပိဋကတ်နှင့် နိကာယ်များကို သီးသန့်စီမဟုတ်ဘဲ ရောနှော ပြဆိုထားပါသည်။

စာရှာလမ်းညွှန်

ပုဒ်တပုဒ်သော်လည်း ၊ နှစ်ပုဒ်နှင့်အထက်သော်လည်းကောင်း ရှာနိုင်ပါသည်။

ရှာသောပုဒ်နှင့်အစတူပုဒ် စာရင်းကို ပုဒ်တပုဒ်ရှာဖွေမှုအတွက်သာ ပြသပါမည်။ နှစ်ပုဒ်နှင့် အထက်အတွက်ကား ပြသပေးမည် မဟုတ်ပါ။ ထိုအစတူ ပုဒ်ကို ရွေးချယ်ပြီးနောက် ကီးဘုတ်ပေါ်မှ စာရှာသော မှန်ဘီလူးခလုပ်လေးကို ဆက်လက်နှိပ်ပြီး ရှာဖွေရပါမည်။

ဖွန့်လမ်းညွှန်

ရွေးချယ်ခွင့် ပေးထားသော ဇော်ဂျီ ၊ ယူနီကုဒ်ဖွန့်သည် ကျမ်းစာ စာသားများနှင့် အဘိဓာန် စာမျက်နှာ ပေါ်၌သာ စာများဖော်ပြရန်အတွက်သာ ဖြစ်ပါသည်။ စာရှာခြင်းစသည်ကိုကား မိမိစက်၏ ဖွန့်အတိုင်း အလုပ်လုပ်ပါသည်။ ဇော်ဂျီဖွန့်သုံးစက်လည်း ယူနီကုဒ်ကို ရွေးချယ်နိုင်သလို ယူနီကုဒ်သမားလည်း ဇော်ဂျီကို ရွေးချယ်နိုင်ပါသည်။

ဇော်ဂျီဖြင့် ပြသရာတွင် သဝေထိုး၊ ရရစ်စသည်တို့ အပေါ်စာကြောင်း၌ ပြတ်မကျန်စေလိုက Fix Line Breaking For Zawgyi Font ကို အမှန်ခြစ်ပေးပါ။ အားသာချက်အနေဖြင့် စာလုံးအဖြတ်အတောက် မှန်ကန်သန့်ရှင်းသည်။ အားနည်းချက်ကား ကော်ပီကူးယူရာတွင် မလိုလားအပ်သော နောက်ကွယ်က မမြင်ရသော စာလုံးတချို့ပါဝင်လာသဖြင့် ဖေ့ဘုတ်စသည်ပေါ် ကူးတင်က အဆင်မပြေဖြစ်တတ်ပါသည်။ ( copy ခလုပ်ကနေ တမျက်နှာလုံး ကူးယူမှသာ အမှန်အတိုင်း ရပါမည်။

ယူနီကုဒ်ဖွန့်ကို ရွေးချယ်ထားပါက် အချို့စာကြောင်းများ၏ အဆုံး၌ ဝိရာမအဖြတ်အတောက်မမှန်ဘဲ ရှိနေတတ်ပါသည်။ (အန်းဒရိုက် webview ၏ အားနည်းချက်ဖြစ်ပါသည်။ )

စာလုံးအကြီးအသေး လမ်းညွှန်

စာလုံးအကြီးအသေးကား ကျမ်းစာ စာသားနှင့် အဘိဓာန် စာသား ဖော်ပြမှု၌သာ သက်ရောက်မှု ရှိပါသည်။

သေးလွန်းသော စာလုံးအရွယ်အစားများကား စခရင်ရှော့ရိုက်လိုရာ၌ အသုံးတည့်ပါလိမ့်မည်။

tab လမ်းညွှန်

tab ကား စာအုပ်များစွာကို တပြိုင်တည်း ဖွင့်၍ ကြည့်ရှုနိုင်သော လုပ်ဆောင်ချက်ဖြစ်ပါသည်။ ဘရောက်ဇာတို့၌ ပါသော tab စနစ်နှင့် အတူတူပင် ဖြစ်ပါသည်။ ဖတ်ဆဲကျမ်းစာ၏ ညာဘက်ထိပ်ရှိ လေးထောင့်ပုံလေးကို နှိပ်ပြီး tab အသစ်ဖွင့်ခြင်း၊ tab ကူးပြောင်းခြင်း၊ မလိုသော( ပိတ်လိုသော) tab ကို ဘယ်ညာသို့ ပွတ်ဆွဲ၍ ပယ်ဖျက်ခြင်း စသည်ပြုလုပ်နိုင်ပါသည်။

ပါဠိ-အဋ္ဌကထာအကူးအပြောင်း လမ်းညွှန်

ညာဘက်အောက်နားရှိ အစက်သုံးစက် ခလုပ်လေးကို နှိပ်၍ ထိုမှတဆင့် မြှားပုံသင်္ကေတကို နှိပ်၍ ပါဠိမှ အဋ္ဌကထာအဖွင့်သို့လည်းကောင်း၊ အဋ္ဌကထာမှ ဋီကာအဖွင့်သို့ လည်းကောင်း ကူးပြောင်းဖတ်ရှုနိုင်ပါသည်။ ထိုအဖွင့်အဋ္ဌကထာ၊ ဋီကာများသည် tab အသစ် အနေဖြင့် ပွင့်လာမည် ဖြစ်ပြီး အဖွင့်ခံ ပါဠိ အဋ္ဌကထာကျမ်းသို့ နောက်ပြန်လှည့်ကြည့်လိုပါက tab button ကို နှိပ်ပြီး အဖွင့်ခံ ကျမ်းကို ပြန်လည်ကြည့်ရှုနိုင်ပါသည်။ အဖွင့်ခံကျမ်း နှင့် အဖွင့်ကျမ်းတို့ tab ထဲတွင် ပါဝင်ပြီးနောက် ထို tab button သုံးပြီး အပြန်အလှန်ကူးပြောင်းကြည့်ရှုနိုင်ပြီးဖြစ်၍ ထိုအခါမျိုး၌ အဆိုပါကျမ်းများအတွက် အကူးအပြောင်းခလုပ်ကို ထပ်မံအသုံးပြုစရာမလိုတော့ပေ။

မှတ်ချက်

စာပိုဒ်နံပါတ်ပေါ်မူတည်၍ ပါ-ဋ္ဌ-ဋီ အကူးအပြောင်းလုပ်ထားသည်ဖြစ်၍ စာပိုဒ်နံပါတ်မပါသော စာမျက်နှာများအတွက် အဖွင့်ကျမ်းကို ကူးပြောင်းဖွင့်နိုင်မည်မဟုတ်ပါ။ အချို့သော ကျမ်းစာများအတွက်လည်း အကူးအပြောင်း စာမျက်နှာ မှန်ကန်အောင် မပြုလုပ်ရသေးပါ။

နိဿယကြည့်ရှုခြင်း လမ်းညွှန်

ညာဘက်အောက်နားရှိ အစက်သုံးစက် ခလုပ်လေးကို နှိပ်၍ ထိုမှတဆင့် စာအုပ်ပုံသင်္ကေတကို နှိပ်၍ နိဿယကျမ်းများကို အလွယ်တကူ ကြည့်ရှုနိုင်ပါသည်။ နိဿယဆော့ဝဲလ်များ သီးသန့်ဒေါင်းလုပ်ပြုလုပ်၍ ထည့်သွင်းထားရပါမည်။

မရှိသေးက ဒေါင်းလုပ်ပြုလုပ်ရန်အတွက် လင့်များပေးထားပါသည်။

နိဿယဆော့ဝဲလ်များကို တကြိမ်မျှ ရှေးဉီးစွာ ဖွင့်ပြီးမှသာလျှင် ပါဠိဆော့ဝဲလ်ကနေ ချိတ်ဆက်ဖွင့်ခြင်းကို ပြလုပ်ပါရန်။ မဟုတ်က ဖွင့်မရပါ။ ဖွင့်မရခဲ့သော် သက်ဆိုင်ရာနိဿယဆော့ဝဲလ်ကို Clear All Data ပြုလုပ်ပေး၍ ထိုနိဿယဆော့ဝဲလ်ကို အရင်ပြန်ဖွင့်ပါ။
